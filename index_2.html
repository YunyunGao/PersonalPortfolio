Create Polygon Masks

This application allows you to create and edit polygon masks on images. The masks can be used for various purposes such as image segmentation, object extraction, or creating cutouts.

With our polygon editor, you can:

  • Draw precise polygon masks on images
  • Edit vertices by double-clicking on a polygon
  • Adjust image properties like brightness, contrast, and gamma
  • Create synchronized polygons across image quadrants

How It Works

Our polygon editor uses Fabric.js to create an interactive canvas where you can draw and manipulate polygon shapes. The editor opens in a popup window, allowing you to focus on your editing task without navigating away from the current page.

To get started, simply click the "Open Polygon Editor" button above and follow these steps:

  1. Click "Draw Mask" to enter drawing mode
  2. Click on the image to place vertices of your polygon
  3. Double-click to complete the polygon
  4. Double-click on a polygon to edit its vertices
  5. Use the sliders to adjust image properties if needed
